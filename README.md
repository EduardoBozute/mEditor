# mEditor
Editor de texto para terminal

-> Como compilar o programa?

    Nota do autor:  o programa tem um executável com todos os módulos, basta executá-lo e o programa será iniciado
    
    Abaixo está o processo de compilação, caso o programa não possa ser executado

    1. Abra o terminal
    2. Navegue até o diretório mEditor
    3. Insira no terminal o comando: gcc tads.c mtext.c index.c -o index.exe
    4. Insira no terminal o comando: ./index.exe

    O programa será compilado.

-> Dúvidas sobre a sintaxe dos comandos?

    Insira 'help' na linha de comando durante a execução do programa

    O programa fará uma amostragem da sintaxe das funções
